3º Mejorar la gestión de categorías
Permite borrar categorías o cambiar su nombre. No se puede borrar una categoría si hay productos con esa categoría. Mostrar el total vendido por cada categoría y el valor actual en almacén.

clase Categoria en models. Habrá que añaidr totalvendido(tbl-lineas_pedidos unidades) y el stock (tbl-productos)

views/categoria/ver.php

SQL:
SELECT * FROM categorias ORDER BY id DESC;"

SELECT c.id, c.nombre, sum(l.unidades) AS 'totalSold', sum(p.stock) AS 'stock'
FROM lineas_pedidos l, productos p, categorias c
WHERE l.producto_id=p.id AND p.categoria_id=c.id
GROUP BY c.id
ORDER BY c.id DESC;
